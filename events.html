Description:  

Attach an event handler function for one or more events to the selected elements.

Syntax:  

on( events [, selector ] [, data ], handler )

var count = 0; $( "body" ).on( "click", "p", function() { $( this ).after( "<p>Another paragraph! " + (++count) + "</p>" ); });
